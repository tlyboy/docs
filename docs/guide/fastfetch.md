# fastfetch

Fastfetch 是一个类似 neofetch 的系统信息获取工具，但速度更快。

## 安装

### macOS (Homebrew)

```sh
brew install fastfetch
```

### Windows (winget)

```sh
winget install fastfetch
```

### Ubuntu (PPA)

```sh
# 添加 Fastfetch PPA
sudo add-apt-repository ppa:zhangsongcui3371/fastfetch

# 更新软件包列表
sudo apt update

# 安装 Fastfetch
sudo apt install fastfetch
```

## 使用

```sh
fastfetch
```
